# ClipboardSharePlugin

[Sailfish](https://sailfishos.org) transfer engine plugin for copying text and url into clipboard.

Fork of the original [ClipboardSharePlugin](https://github.com/Maledictus/harbour-clipboardshareplugin) to support Sailfish >= 4.4.

## Install

Install via [Chum](https://github.com/sailfishos-chum/main).
